Pill label chip — statuses ("✓ CLEAR", "READY", "LOCKED"), rewards ("+ RADAR DISH · radar()"), counts ("★ 13/15"). ALL-CAPS text.

```jsx
<Chip color="green">✓ CLEAR</Chip>
<Chip color="amber" dashed>REWARD: TURBO TREADS</Chip>
<Chip>LOCKED</Chip>
```

Color = meaning (see readme color roles). Dashed always means "not yet" — locked, future, or hint.
